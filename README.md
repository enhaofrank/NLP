## 舆情分析
舆情分析 或情感分析， 在自然语言处理领域是个主要的应用方向， 常见的情况是为某个公司或某个产品，根据舆情数据，分析出消费者的情感倾向，比如：根据商品评论数据（比如华为P9），分析用户情感的倾向，正，负，中性。
